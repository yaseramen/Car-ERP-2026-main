## تدفق البيانات الرئيسي

### 1. تركيب قطعة في الورشة
```
repair_order_items (إضافة) 
  → stock_movements (خصم من المخزن - movement_type: workshop_install)
  → item_warehouse_stock (تحديث الكمية)
  → invoice_items (إضافة للفاتورة)
```

### 2. إصدار فاتورة
```
invoices (إنشاء)
  → digital_service_fee = max(0.5, total * 0.0001)
  → wallet_transactions (خصم من محفظة الشركة - type: digital_service)
  → invoice_payments (المدفوعات)
  → treasury_transactions (إضافة للخزينة)
```

### 3. مرتجع
```
invoices (is_return=1, original_invoice_id=الفاتورة الأصلية)
  → stock_movements (إعادة للمخزن)
  → treasury_transactions (خصم من الخزينة)
```

### 4. بحث OBD
```
obd_searches (إنشاء)
  → obd_codes (بحث محلي أولاً)
  → إذا لم يوجد: استدعاء AI وتخزين
  → wallet_transactions (خصم 1 جنيه - type: obd_search)
```
